# Prosthetic Controlled Drone

## Overview
This project presents a drone controlled using prosthetic signals, allowing users to interact with the system through physical inputs.

## Motivation
The goal of this project is to explore human-machine interaction and improve accessibility using embedded systems.

## System Architecture
- Input: Prosthetic signals (sensors)
- Processing: Microcontroller (Arduino)
- Output: Drone movement control

## Features
- Real-time signal processing
- Stable drone control
- Embedded system integration

## Technologies Used
- Arduino
- C/C++
- Sensors (mention type if possible)

## Challenges
- Signal noise filtering
- Stability of drone control
- Calibration issues

## Results
The system was able to successfully translate physical signals into drone movements with acceptable accuracy.
